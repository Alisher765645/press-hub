# PRESS HUB

Единый центр пресс-службы нефтегазовой компании — внутренняя веб-платформа для централизованного управления коммуникационной деятельностью головной компании и её 24 дочерних предприятий.

Прототип реализован как статическое веб-приложение без серверной части: HTML5 + CSS3 + Vanilla JavaScript, хранение данных в `localStorage`, инфографика на HTML5 Canvas.

## Возможности

- **Контент-план (календарь)** — месячная сетка событий по 24 предприятиям, фильтры по предприятию и типу материала, виджет предстоящих событий, сводная статистика.
- **Архив материалов** — карточное хранилище с поиском и фильтрами, просмотр полного текста, передача материала в ИИ-ассистент.
- **ИИ-ассистент** — 5 режимов (выжимка, аналитика, пресс-релиз, контент-план, инфографика), загрузка файлов, сохранение результатов в архив, генерация PNG-инфографики.

Все данные сохраняются между сессиями в браузере.

## Запуск

Откройте `index.html` в браузере. Локально достаточно дважды кликнуть по файлу либо поднять статический сервер:

```bash
python -m http.server 8000
# затем откройте http://localhost:8000
```

## Настройка ИИ

ИИ-функции обращаются к Anthropic Claude API. Приложение не хранит ключ в коде. Откройте кнопку **⚙ Настройки** и укажите один из вариантов:

- **URL прокси** (рекомендуется) — серверный эндпоинт, проксирующий запросы к `https://api.anthropic.com/v1/messages`. Ключ остаётся на сервере.
- **API-ключ Anthropic** — прямой доступ из браузера для обучения/прототипа (включает заголовок `anthropic-dangerous-direct-browser-access`). Не используйте этот режим в продакшене.

Значения хранятся только локально в браузере.
